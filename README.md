# vue-todo-app
这是个todo app，基于Vue.js，脚手架为Vue CLI 3.0.

练习用。

样式以及原型参照：http://todomvc.com/examples/vue/#/all

用到了custom directive。

遗留问题：

Vue如何做到即时更新


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
